# ***\*一、Topic命名规范\****

每个电厂独立Topic，命名格式如下：

| **电厂** | **编码** | **Topic名称** |
| -------- | -------- | ------------- |
| 黄岛电厂 | sd02001  | gyhly_sd02001 |
| 东营电厂 | sd03001  | gyhly_sd03001 |
| 郓城电厂 | sd04001  | gyhly_sd04001 |
| 鲁北电厂 | sd05001  | gyhly_sd05001 |
| 滨州电厂 | sd06001  | gyhly_sd06001 |
| 临清电厂 | sd07001  | gyhly_sd07001 |

命名规则：gyhly_{plant_code}，全小写，下划线连接。
